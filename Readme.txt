Oppgave 1: 
Forskjellen mellom CSS ID og Class Elementer.
Class elementer gir deg muligheten til å gi dine HTML-elementer forskjellig utseende (fonter, størrelse, farger osv.) For eksempel om man ønsker at 
en tekst skal være hvit og stor den ene gangen, og liten og svart en annen gang, så kan dette gjøres ved hjelp av Class elementer.
Når det kommer til CSS ID er det mange likheter med Class elements. 
CSS ID er på mange måter en unik id til et element. Forskjellen her mellom disse to er at Class elements kan bli brukt opp til flere ganger,
mens CSS ID kun kan brukes en gang.
Derfor er det viktig å være bevisst på hva du skal gjøre, hvis du for eksempel skal gi alle headingene dine samme utseende kan det 
være fordelaktig å bruke Class elements. 
CSS ID er ment for en spesiell handling eller element, men Class elements kan bli brukt til flere handlinger.

Oppgave 2: "Vurder og sammenlign brukbarhet til versjoner i deloppgave 0 og deloppgave 1 av designet
for presentasjon av en liste"
Først versjonen i del 0, jeg vurderer her brukbarheten til lite god. En liste med over 100 objekter i forskjellige farger er

Derfor mener jeg at versjonen i deloppgave 1 er den mest brukbare versjonen. Her vil en rullegardin virke mer oversiktlig og 
designet vil være mer appelerende. 
Designet er viktig for at brukeren skal tilfredsstilles, der mener jeg at versjon 0 har et langt stykke å gå. 


Oppgave 4: Hovedforskjellen mellom <div> og <span> elementer er at <div>-taggen lager en seksjon i et HTML dokument. 
<div> taggen brukes til å grupere "bolker" eller koder for å tilegne dem CSS.
En <span> tag brukes på sin side som en "hook" til en del av en tekst. For eksempel kan <span> brukes til å endre farge på et 
bestemt ord i en setning. Når et slikt ord er "hooket" av span kan dette ordet endres med javascript eller CSS.
Så hovedforskjellen er at Div taggen tar for seg hele setninger eller avsnitt, mens span brukes til mindre og mer konkrete ord
eller korte setninger.
